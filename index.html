接続設定 ▸
合言葉トークンを知っている人はこのURLで読み書きできます。長く・秘密に保ち、本当に機微な顧客情報は中継に載せない運用を。データはGAS側の保持時間を過ぎると自動削除されます。
初回セットアップの手順 ▸

① バックエンド（GAS）

  1. Googleドライブで新しいスプレッドシートを作成
  2. 「拡張機能」→「Apps Script」を開き、Code.gs を全部貼り付け
  3. TOKEN を自分だけの長い合言葉に書き換えて保存
  4. 「デプロイ」→「新しいデプロイ」→ 種類=ウェブアプリ / 実行=自分 / アクセス=全員
  5. 発行された /exec URL を、上の「接続設定」に貼る
GASのコードを直したら、保存だけでは反映されません。「デプロイを管理」→ ✏️ → バージョン「新バージョン」→ デプロイ、で更新（URLは不変）。

② iPhone（ショートカット） — App不要。2つ作ります。

PCへ送る: 「クリップボードを取得」→「テキスト」で JSON を組む→「URLの内容を取得」で上のURLにPOST。
PCから受け取る: 「URLの内容を取得」でGET→ 返ってきた text を「クリップボードにコピー」。
具体的なアクションの並びはチャット側の手順をご覧ください。
